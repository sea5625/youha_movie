## Getting Started 

Project clone & install

```sh
$ git clone https://github.com/sea5625/youha_movie.git # clone
$ cd youha_movie 

# yarn 
$ yarn install

# npm 
$ npm install
```

Start

```sh
# yarn 
$ yarn start

# npm 
$ npm start
```
